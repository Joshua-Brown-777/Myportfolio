# Assets Folder

This folder contains images and other static assets for your portfolio website.

## How to add images:

1. Upload your image files to this folder
2. Name them descriptively (e.g., `project1.jpg`, `ecommerce-screenshot.png`, etc.)
3. Let the assistant know the file names so the code can be updated to import them

## Supported formats:
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.svg`

## Example file structure:
```
/assets
  ├── project1.jpg
  ├── project2.png
  ├── project3.webp
  └── README.md
```
